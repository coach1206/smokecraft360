Build a Verified Progression, Personal Humidor, Competition, and Status System for SmokeCraft 360.

GOAL:
Create a system where users can ONLY level up through real, verified purchases at lounges, track their personal cigar/drink history, compete for status, and unlock elite rewards like creating their own cigar and band.

--------------------------------------------------
1. VERIFIED ORDER SYSTEM (CORE)

Extend orders table:

- id
- userId
- venueId
- productId
- pairingId
- foodId
- orderType
- status ("pending", "completed")
- verified (boolean)
- verifiedAt (timestamp)
- verificationMethod ("staff", "qr", "pos")

--------------------------------------------------
2. VERIFICATION METHODS

Support:

A. STAFF CONFIRMATION
- staff marks order completed → verified = true

B. QR CODE
- generate QR per order
- scan at lounge → verified = true

C. POS (future ready)
- auto-verify via POS

--------------------------------------------------
3. XP SYSTEM (STRICT)

ONLY award XP when:
verified = true

NO verification = NO XP

--------------------------------------------------
4. XP REWARDS

Award XP:

- cigar purchase → +10
- drink pairing → +8
- full combo (cigar + drink + food) → +20 bonus
- new product tried → +5 bonus

--------------------------------------------------
5. PERSONAL HUMIDOR (REAL HISTORY)

Create:

userHumidor:

- id
- userId
- productId
- category
- quantityPurchased

ONLY update if:
verified = true
