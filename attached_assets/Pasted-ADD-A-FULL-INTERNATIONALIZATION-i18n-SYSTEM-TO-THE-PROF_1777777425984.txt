ADD A FULL INTERNATIONALIZATION (i18n) SYSTEM TO THE PROFOUND INNOVATION (P.I) PLATFORM THAT ENABLES MULTI-LANGUAGE SUPPORT ACROSS ALL USER-FACING INTERFACES WITHOUT MODIFYING CORE FUNCTIONALITY OR BREAKING EXISTING SYSTEMS

---

🚨 CORE OBJECTIVE

Allow users to:

- switch languages dynamically
- have all UI text update instantly
- persist language preference across sessions
- support multiple countries and regions

---

🧱 FRONTEND IMPLEMENTATION (REACT)

INSTALL:

i18next  
react-i18next  

---

CREATE FOLDER:

/client/i18n/

FILES:

i18n.js  
en.json  
es.json  
fr.json

🧠 LANGUAGE FILE STRUCTURE

Example (en.json):

{
  "intro_title": "Craft Your Experience",

}

---

CREATE TRANSLATIONS:

- English (default)
- Spanish
- French

(Expandable later)

---

🔥 i18n SETUP

In i18n.js:

- initialize i18next
- load language files
- set default language = "en"
- enable browser language detection
- fallback to English

---

💻 APPLY TRANSLATIONS

Replace ALL hardcoded text with:

const { t } = useTranslation()

Example:

<h1>{t("intro_title")}</h1>

---

🌐 LANGUAGE SWITCHER COMPONENT

CREATE:

/client/components/LanguageSwitcher.jsx

Features:

- dropdown or buttons (EN / ES / FR)
- on click → change language

Example:

i18n.changeLanguage("es")

---

💾 LANGUAGE PERSISTENCE

Store selection:

localStorage.setItem("pi_language", lang)

On app load:

- read saved language
- apply automatically
