Build an Experience Control Panel for the existing CraftHub / Axiom immersive swipe system.

Do not rebuild the swipe engine.
Do not rebuild the realism layer.
Do not create duplicate routes.

Create a protected admin route:

/admin/experience-control

Purpose:
Allow venue owners, managers, and super_admin users to tune the immersive environment without editing code.

Controls required:

1. Atmosphere Intensity
Range: 0 to 100
Controls glow strength, background brightness, and cinematic depth.

2. Particle Density
Range: 0 to 100
Controls embers, vapor, bubbles, condensation, dust, and foam details.

3. Motion Calmness
Range: 0 to 100
Higher value means slower, smoother, calmer movement.
Lower value means more energetic motion.

4. Reveal Pacing
Range: 0 to 100
Controls how dramatic or fast the RevealPage animation feels.

5. Sound Volume
Range: 0 to 100
Controls ambient and interaction sound levels through soundEngine.ts.

6. Performance Mode
Options:
cinematic
balanced
low-power

cinematic = maximum atmosphere
balanced = default
low-power = reduced particles and lower motion for tablets/kiosks

7. Craft Overrides
Allow per-craft override for:
SmokeCraft
PourCraft
BrewCraft
VapeCraft

Each craft can override:
atmosphereIntensity
particleDensity
motionCalmness
revealPacing
soundVolume
performanceMode

Database:
Create table experience_control_settings:

id
venue_id nullable
craft_type nullable
atmosphere_intensity int default 70
particle_density int default 65
motion_calmness int default 55
reveal_pacing int default 70
sound_volume int default 40
performance_mode text default 'balanced'
created_at timestamp default now()
updated_at timestamp default now()
