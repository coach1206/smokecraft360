🔐 AUTH + RBAC SYSTEM

- JWT login route: POST /api/auth/login
- middleware to verify token
- role-based access:

super_admin → full access  
venue_admin → own venue only  
staff → limited access  

---

💳 STRIPE PAYMENT SYSTEM

POST /api/payments/create-intent

- fetch cigar + whiskey from DB
- calculate price (base + markup)
- apply idempotency key
- create PaymentIntent
- return clientSecret

---

🔔 STRIPE WEBHOOK (SOURCE OF TRUTH)

POST /api/webhooks/stripe

Handle:

payment_intent.succeeded:

- validate signature
- update ORDER status → paid
- decrement inventory
- create COMMISSION record

---

💰 STRIPE CONNECT PAYOUT SYSTEM

POST /api/stripe/connect

- create or attach Stripe Express account
- store stripe_account_id in VENUES

POST /api/payouts/request

- create payout request
- enforce minimum threshold

POST /api/payouts/approve (admin only)

- send transfer via stripe.transfers.create()
- store stripe_transfer_id
- update status
