3. PRODUCTS API

GET /api/products?type={type}

- read query param
- fetch from db
- return JSON array

---

4. SCORING ENGINE

POST /api/scoring

Input:

{
  flavor: number,
  strength: number,
  pairing: number
}

Logic:

score = (flavor * 0.4) + (strength * 0.3) + (pairing * 0.3)

Return:

{
  score: number,
  label: string
}

---

5. ORDER ENDPOINT

POST /api/order

Input:

{
  productIds: [],
  theme: string
}

Return:

{
  status: "order_created",
  message: "Order received"
}

(No payment processing yet — just structure)
