2. CLICK + SWIPE SOUND SYSTEM

Add real tactile feedback.

---

SOUND FILES (add to /public/sounds/):

* click.mp3 (light tap)
* swipe.mp3 (soft whoosh)
* select.mp3 (premium confirmation)

---

JS IMPLEMENTATION

```js
const clickSound = new Audio('/sounds/click.mp3');
const swipeSound = new Audio('/sounds/swipe.mp3');
const selectSound = new Audio('/sounds/select.mp3');

export function playClick() {
  clickSound.currentTime = 0;
  clickSound.play();
}

export function playSwipe() {
  swipeSound.currentTime = 0;
  swipeSound.play();
}

export function playSelect() {
  selectSound.currentTime = 0;
  selectSound.play();
}
```

---

TRIGGER RULES

* Tap → click sound
* Swipe → swipe sound
* Final selection → select sound

3. SMOOTH MOTION SYSTEM (MANDATORY)

Cards must NEVER snap instantly.

---

CSS

```css
.card {
  transition: transform 0.35s ease, opacity 0.35s ease;
  will-change: transform;
}
```

---

JS DRAG LOGIC (concept)

```js
onDrag = (x) => {
  setTransform(`translateX(${x}px) rotate(${x * 0.05}deg)`);
}

onRelease = (x) => {
  if (x > 120) accept();
  else if (x < -120) reject();
  else reset();
}
```
