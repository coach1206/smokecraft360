The Universal Experience Brain (ExperienceEngine.tsx)
This "wires" the state management for all four crafts. It ensures that if a guest is building a cigar in SmokeCraft, the EEIS staff layer knows exactly which flavor profile is being selected in real-time.

JavaScript
import React, { createContext, useContext, useReducer } from 'react';

// Central Intelligence State
const initialState = {
  activeModule: 'portal', // smoke, pour, brew, vape
  guestProfile: { name: '', preferences: [], history: [] },
  sessionData: { currentRound: 1, score: 0, pairings: [] },
  inventoryStatus: 'synced',
  eeisMode: false,
};

const ExperienceContext = createContext();

export const ExperienceProvider = ({ children }) => {
  const [state, dispatch] = useReducer((state, action) => {
    switch (action.type) {
      case 'SET_MODULE': return { ...state, activeModule: action.payload };
      case 'UPDATE_PROFILE': return { ...state, guestProfile: { ...state.guestProfile, ...action.payload } };
      case 'TOGGLE_EEIS': return { ...state, eeisMode: !state.eeisMode };
      case 'SOVEREIGN_PURGE': return initialState; // Absolute reset for JC's security
      default: return state;
    }
  }, initialState);

  return (
    <ExperienceContext.Provider value={{ state, dispatch }}>
      {children}
    </ExperienceContext.Provider>
  );
};
2. The Touch-Responsive Interaction Wire
This refactors your buttons into Tactile Control Surfaces. Since you are targeting Android first, we use onTouchStart to eliminate the 300ms "web lag".

JavaScript
const TactileCard = ({ title, onTrigger }) => {
  return (
    <div 
      className="relative group overflow-hidden rounded-xl border border-gold/20 bg-black/40 backdrop-blur-md"
      onTouchStart={(e) => {
        e.currentTarget.style.transform = 'scale(0.96)';
        e.currentTarget.style.borderColor = 'rgba(212, 175, 55, 0.8)';
      }}
      onTouchEnd={(e) => {
        e.currentTarget.style.transform = 'scale(1)';
        e.currentTarget.style.borderColor = 'rgba(212, 175, 55, 0.2)';
        onTrigger(); // Launches the Craft Module
      }}
    >
      <div className="p-8">
        <h2 className="text-gold tracking-[0.3em] font-bold uppercase">{title}</h2>
        <p className="text-silver text-xs mt-2 opacity-60">INITIALIZE RITUAL</p>
      </div>
    </div>
  );
};
