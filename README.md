# SmokeCraft 360: Interactive Master Experience

## Project Overview
**SmokeCraft 360** is a high-fidelity, touch-optimized hospitality orchestration platform designed for luxury lounge environments. It provides a cinematic journey from initial system power-up to staff-handoff concierge services.

## Core Features
- **Cinematic Initiation**: A multi-stage startup sequence featuring "Profound Innovations" and "NOVEE OS" branding.
- **CraftHub Portal**: A central hub for selecting curated craft experiences (Smokecraft, Pourcraft, Brewcraft, Winecraft).
- **Interactive Smoke Engine**: Real-time vanilla JavaScript smoke simulations for atmospheric immersion.
- **Touch-Optimized UI**: Precision-engineered for kiosks and tablets with large tap targets and tactile feedback.
- **Staff-Guest Ecosystem**: Integrated "E.A.T. System" (Environment, Asset Transactions) for seamless concierge service.

## Experience Flow (S1–S6)
1.  **Stage 1: Immersion**: Entry point into the luxury atmosphere.
2.  **Stage 2: Identity**: Personalization and guest profiling.
3.  **Stage 3: Leaf Matrix**: Educational exploration of premium tobacco and craft materials.
4.  **Stage 4: Reveal & Handoff**: Transition from automated exploration to personalized service.
5.  **Stage 5: Concierge Service (Guest)**: On-demand requests and service tracking.
6.  **Stage 6: E.A.T. Dashboard (Staff)**: Backend orchestration for lounge management.

## Deployment Instructions

### Local Preview
1. Download the `SmokeCraft_360_Master.html` file.
2. Open the file in any modern web browser (Chrome, Safari, or Edge recommended).

### GitHub Deployment
1. Create a new repository on [GitHub](https://github.com).
2. Upload the `.html` file and any exported image assets.
3. Navigate to **Settings > Pages**.
4. Set the **Branch** to `main` and the **Folder** to `/(root)`.
5. Click **Save**. Your experience will be live at the provided URL.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript.
- **Motion**: CSS Keyframes & DOM-based Particle Systems.
- **Design System**: Hospitality OS (Dark Mode / Playfair Display).
